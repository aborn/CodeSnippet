dredis
==========

### INTRO
dredis is a distributed redis system framework, which based on the
[phpredis](https://github.com/nicolasff/phpredis),
[hiredis](https://github.com/redis/hiredis) and
[rediska](http://rediska.geometria-lab.net/).
